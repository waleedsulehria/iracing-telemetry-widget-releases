# Release Notes - v1.0.0

**Release Date:** 2025-12-22

## What's New in v1.0.0

### Features
- Real-time steering, throttle, brake visualization
- Lap time comparison with reference laps
- Gear and RPM display
- Brake bias display
- TC & ABS levels display
- Real-time ABS activation indicator
- Brake comparison graphs

## Installation

1. Download `iracing_telemetry.exe`
2. Run the executable (no installation needed)
3. Start iRacing and begin your session
4. The overlay will automatically connect

## System Requirements

- Windows 10 or Windows 11
- iRacing (active subscription required)
- 4GB RAM minimum (8GB recommended)
- ~100MB free disk space

## Notes

- All dependencies are bundled in the executable
- No Python installation required
- The app automatically detects when iRacing is running
