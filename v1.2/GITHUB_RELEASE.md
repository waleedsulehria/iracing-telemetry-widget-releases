### Track Markers Setup

1. **Open Settings** (âš™ï¸ icon in top-left)
2. **Scroll to "Track Markers"** section
3. **Set hotkeys:**
   - **Green button** = Mark position (e.g., `Alt+M` or `joy0_btn12`)
   - **Orange button** = Clear closest marker
   - **Red button** = Clear all markers
4. **Adjust warning distance** slider (default: 50m)
5. **Drive and mark!** Press your hotkey to set markers

- Driver Info (name, number, logo)

- Steering Wheel
- Lap Times

- Driver Controls (BB, TC, ABS)
